# Antivirus Simulation (NTP Project)

Bu proje, Nesne Tabanlı Programlama dersi kapsamında geliştirilmiş
**imza tabanlı antivirüs simülasyonudur**.

❗ Gerçek virüs içermez.

## Amaç
- Dosya tarama mantığını göstermek
- Zararlı dosyaları tespit edip karantinaya almak
- NTP kavramlarını uygulamak

## Çalışma Mantığı
- Dosyalar taranır
- Bilinen imzalarla karşılaştırılır
- Zararlı bulunan dosyalar **Quarantine** klasörüne taşınır
- Sonuçlar konsola yazdırılır

## Kullanılan Yapılar
- Class / Interface
- Enum
- Dosya okuma & taşıma
- Exception handling

## Çalıştırma
- Projeyi aç
- `Program.cs` çalıştır
- Enter’a bas
